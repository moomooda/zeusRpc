# zeusRpc

本项目是一个轻量级RPC框架，也是对SPI的一次实践，通过SPI可自定义扩展通信模块、服务管理中心、负载均衡算法。对于这三个部分，本项目分别提供的默认实现是Netty、ZooKeeper、一致性Hash算法。
